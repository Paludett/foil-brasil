# 🌟 Minha Jornada — AMF & FOIL

Site de linha do tempo criado para o concurso da FOIL BRASIL na AMF.

---

## 📖 Sobre o Projeto

Este projeto é um site de linha do tempo com abordagem mobile-first, desenvolvido em aproximadamente 2 horas para um concurso interno da Antonio Meneghetti Faculdade (AMF). O prêmio da competição é uma viagem para Mato Grosso do Sul. O conteúdo apresenta a história pessoal da evolução pessoal na faculdade desde a chegada à AMF, passando por 6 marcos principais entre dezembro de 2024 e fevereiro de 2026.

## ✨ Funcionalidades

- Design elegante inspirado na identidade visual da AMF
- Linha do tempo alternada esquerda/direita com animações de scroll
- Fotos reais em cada card com overlay dourado
- Totalmente responsivo para celular
- Tipografia refinada (Dancing Script, Cinzel, Cormorant Garamond)
- Paleta de cores cream e dourado fiel à identidade da faculdade

## 🛠️ Tecnologias

- HTML5
- CSS3 (animações, Intersection Observer API)
- JavaScript vanilla
- Google Fonts

## 🚀 Como rodar

Este projeto é HTML puro, sem dependências e sem etapas de build. Basta abrir o arquivo `index.html` no navegador ou publicar a pasta em serviços como Vercel ou Netlify.
